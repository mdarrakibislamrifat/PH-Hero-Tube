Scope Of var:

1.Variables declared with var are in the function scope.

2.In the var tab, when you will run the code, you will see an error as a is not defined because var variables are only accessible in the function scope.

3.In the var tab, when you run the code, you will see there is no error and that we can declare var variables after defining them.

4.In the var tab, when you run the code, you will see that there is no error as we allowed to declare the same variable again

Scope of Let:

1.Variables declared as let are in the block scope.

2.In the let tab, when you run the code, you will see an error as i is not defined because let variables are only accessible within the block they are declared.

3.In the let tab, when you run the code, you will get an error as let variables do not support hoisting.

4.In the var and let tab, when you run the code, you will see that there is no error and we can define new values to the var and let variables.

Scope of Const:

1.Variables declared as const are in the block scope.

2.In the const tab, when you run the code, you will see that x was defined under the block was accessible, but when you try to access x outside that block, you will get an error.

3.In the const tab, when you run the code, you will get an error as const variables do not support hoisting

4.In the const tab, when you run the code, you will get an error as a const variables value cannot be reassigned.


Null and Undefined

Undefined means the variable has been declared, but its value has not been assigned. Null means an empty value or a blank value. The typeof() operator returns undefined for an undefined variable. The typeof() operator returns the type as an object for a variable whose value is assigned as null.


Rest api

REST API stands for Representational State Transfer Application Programming Interface. It is a set of architectural principles and constraints that are commonly used in web development to design networked applications. REST APIs are used to enable communication between different software systems over the internet.
